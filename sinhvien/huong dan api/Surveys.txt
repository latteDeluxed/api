    GET /surveys.php - Get all active surveys (current date between start and end)

    GET /surveys.php?survey_id=X - Get single survey with response count

    GET /surveys.php?active=1 - Get all active surveys

    GET /surveys.php?active=0 - Get all inactive surveys

    GET /surveys.php?upcoming - Get upcoming surveys (start_date > now)

    GET /surveys.php?past - Get past surveys (end_date < now)

    POST /surveys.php - Create new survey

    PUT /surveys.php?survey_id=X - Update survey

    DELETE /surveys.php?survey_id=X - Delete survey (only if no responses)
