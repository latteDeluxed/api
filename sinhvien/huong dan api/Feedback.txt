    GET /feedback.php?feedback_id=X - Get single feedback response

    GET /feedback.php?survey_id=X - Get all feedback for a survey

    GET /feedback.php?survey_id=X&student_id=Y - Get specific student's feedback for a survey

    GET /feedback.php?student_id=X - Get all feedback submissions by a student

    POST /feedback.php - Submit new feedback

    DELETE /feedback.php?feedback_id=X - Delete feedback (admin only)
